This is the sample code to generate synthetic IMU data from MoCap motion sequence in SMPL format, as done in [1]. 

To run this code please:
1, Download SMPL python2 module from [2] and place the folder 'SMPL_python_v.1.0.0' in this path;
2, Download the MoCap sequence from [3]. Here one sample file from HumanEva is used in this code;
3, Run run_demo.sh.

Reference
1, https://dip.is.tue.mpg.de; https://ait.ethz.ch/projects/2018/dip/
2, https://smpl.is.tue.mpg.de/
3, https://amass.is.tue.mpg.de/
